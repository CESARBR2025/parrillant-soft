| Nombre                  | Hex       | Uso en la imagen                        |
| ----------------------- | --------- | --------------------------------------- |
| Fondo oscuro principal  | `#1a1f35` | Fondo general de la página              |
| Fondo con tinte púrpura | `#1e2340` | Degradado del fondo (tono medio)        |
| Fondo card / overlay    | `#252b45` | Tarjeta central con texto               |
| Naranja principal       | `#e8520a` | Texto destacado, botón CTA primario     |
| Naranja hover / badge   | `#d44d0f` | Punto del badge superior                |
| Blanco puro             | `#ffffff` | Títulos principales                     |
| Blanco suave            | `#e8eaf0` | Texto de cuerpo y párrafos              |
| Gris medio              | `#9ba3b8` | Texto secundario / muted                |
| Borde card              | `#2e3555` | Borde sutil de la tarjeta               |
| Botón secundario fondo  | `#2a3050` | Fondo del botón "Entender la propuesta" |
| Botón secundario borde  | `#3d4570` | Borde del botón secundario              |

**Tip de uso:**

- Usa `#1a1f35` como `background` base del `body`
- Usa `#e8520a` como tu `--color-primary` o `accent`
- Usa `#ffffff` y `#e8eaf0` para jerarquía tipográfica
- Usa `#252b45` para cards, modales o secciones elevadas
